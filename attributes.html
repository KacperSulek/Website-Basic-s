Attribute

Alle HTML Elemente können attribute beinhalten.

Attribute geben zusätzliche informationen über das Element.

Attribute sind immer am anfang des Tags

Attribute kommen meistens immer mit dem namen und einen wert wie: name="wert"

href

Der href attribut spezifisiert die URL und wird da hin weitergeleitet

src

Der src attribut spezifisiert den pfad von wo das bild kommt was angezeigt wird

Es können ganze URLs eingeben werden oder nur relative URL also URLs die nur ein paar Pfade zeigt

width / height

width und height definieren die höche und die breite eines bilds in pixeln

alt

Der alt attribut ist dafür zuständig das falls das bild nicht gefunden werden kann eine altenative nachricht eingeben kann oder einen error schreiben kann

style

Style attribut ist dafür zuständig das element mit farbe, schriftart, größe, und viel mehr zu geben

lang

Der lang attribut soll immer dazugegeben werden da es die suchmaschinen dabei hilf die seite zu finden

title

Der wert von attribut title wird angezeigt wenn man mit der maus darüber zeigt auf das element
